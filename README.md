# Omniauth::MrcTrain

MRC Train SSO

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-mrc-train'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-mrc-train
